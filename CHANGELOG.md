## [0.3.0] - 2026-07-15

### Added

- 音乐播放器新增自动连播、随机推荐和单曲循环控制。
- 音乐播放时，桌面浮动球显示四柱律动状态。
- 时间回顾新增鼠标点击次数、键盘按键次数和最近一次键鼠输入时间。
- 托盘和浮动球新增原生右键菜单，可显示主窗口、隐藏浮球或完全退出 DayMate。

### Changed

- 电脑活跃时间使用 Windows 最近键鼠输入判断，连续 5 分钟无输入后不再把亮屏时间计为活跃。
- 活动数据库迁移至 v2，为每段会话保存键鼠次数汇总。

### Fixed

- 修复鼠标移动到托盘图标上时主窗口自动弹出的问题；现在只有托盘左键单击才打开窗口。
- 修复应用只能通过任务管理器退出的问题。

### Security

- 键鼠统计只保存次数，不记录按键内容、鼠标位置或输入目标。

## [0.2.0] - 2026-07-14

### Added

- 新增应用内智能音乐播放器，根据时间和待办状态从 Audius 大型开放音乐目录推荐并播放音乐。
- 新增四首 CC0 离线音乐、六类音乐偏好、开放曲库搜索和本地歌曲导入；断网时仍可播放。
- 新增 AI 音乐类别推荐，只发送活跃分钟、未完成任务数和音乐偏好等最小化汇总。
- 新增七套每日舒适背景，同一天保持一致，并支持用户手动切换。
- 新增多 AI 服务商设置：商汤日日新、OpenAI、DeepSeek、通义千问、智谱、Moonshot、OpenRouter、Ollama 与自定义 OpenAI 兼容接口。
- 新增 AI 接口连接测试和按服务商独立保存密钥。

### Changed

- 每日音乐由跳转外部平台改为在 DayMate 内直接播放。

### Fixed

- 修复音乐搜索偶尔只返回图片、导致“音乐暂时走丢了”的问题。
- 修复切换音乐类别时每日好句也被意外更换的问题。
- 修复切换 DayMate 页面或收起主窗口后音乐停止的问题，播放内核现在常驻应用生命周期。

### Security

- API Key 使用 Windows 凭据管理器保存，不写入 Zustand、SQLite、日志、Git 仓库或安装包。
- 远程自定义 AI 接口强制使用 HTTPS，仅允许本机 Ollama 使用 HTTP。

## [0.1.1] - 2026-07-14

### Added

- 新增可拖动的桌面浮动球，点击即可恢复 DayMate 主窗口。
- 设置页新增桌面浮动球开关，侧边栏新增“收起为浮球”快捷操作。
- README 新增真实应用界面截图。
- 新增 DayMate 正式品牌图标，并用于桌面、安装包和托盘。

### Changed

- GitHub Release 改为版本标签推送后自动公开发布 Windows 安装包。

## [0.1.0] - 2026-07-13

### Added

- 首次启动引导、今日首页、任务管理和专注倒计时。
- 基于截止日期、优先级和预计时间的本地任务推荐。
- 每日好句、音乐、轻松内容和微挑战。
- Windows 前台应用与空闲时间采集，SQLite 本地会话存储。
- 活动记录、窗口标题和空闲检测隐私开关。
- 时间回顾、系统托盘、深浅主题和本地偏好保存。
- 前端测试、版本一致性检查、GitHub CI 与 Release 工作流。

### Security

- 窗口标题默认关闭。
- 不采集键盘输入、屏幕截图、聊天正文或文档正文。
- 活动记录默认不上传。
