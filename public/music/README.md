# Bundled music credits

These fallback tracks are bundled so DayMate can play music without a network connection. All are published under CC0 on OpenGameArt:

- `orient8.ogg` — “Oriental8” by Tozan: https://opengameart.org/content/oriental8
- `not-that-east.ogg` — “Not That East” by KiluaBoy: https://opengameart.org/content/sci-fi-adventure-eastern-quiet-piano-loop
- `calm-theme.ogg` — “calm theme” by pebonius: https://opengameart.org/content/calm-theme
- `oriented.ogg` — “Oriented” by yd: https://opengameart.org/content/oriented

License: CC0 1.0 Universal. Attribution is not required, but DayMate keeps the source links visible in the player.
