# DayMate 日伴使用手册

> 面向从 GitHub 下载和使用 DayMate 的普通 Windows 用户。无需安装 Node.js、Rust，也无需注册账号。

## 立即下载

[⬇️ **直接下载 DayMate 0.3.2 Windows 安装包（64 位）**](https://github.com/zhangweiguo9719-web/daymate-desktop/releases/download/v0.3.2/DayMate_0.3.2_x64-setup.exe)

- [查看最新版本与更新说明](https://github.com/zhangweiguo9719-web/daymate-desktop/releases/latest)
- [下载 SHA256 校验文件](https://github.com/zhangweiguo9719-web/daymate-desktop/releases/download/v0.3.2/SHA256SUMS.txt)
- 当前支持：Windows 10 / Windows 11，64 位
- GitHub 账号：不需要
- AI API Key：不需要；只有 AI 增强功能需要自行配置

以后发布新版本时，请优先进入“查看最新版本与更新说明”页面下载最新安装包。

## 1. 安装 DayMate

1. 点击上方“直接下载”，保存 `DayMate_0.3.2_x64-setup.exe`。
2. 双击安装包，按照安装向导完成安装。
3. 从桌面快捷方式或开始菜单打开“DayMate 日伴”。
4. 第一次启动时完成昵称、身份、陪伴语气、隐私权限和首个任务设置。

Windows 可能因为应用仍处于早期发布阶段而显示 SmartScreen 提示。如果安装包确实来自本仓库的 GitHub Releases，可选择“更多信息”后检查发布者和文件名，再决定是否运行。建议同时使用 Releases 中的 `SHA256SUMS.txt` 校验文件完整性。

### 校验安装包（可选）

在安装包所在文件夹打开 PowerShell，执行：

```powershell
Get-FileHash .\DayMate_0.3.2_x64-setup.exe -Algorithm SHA256
```

将输出结果与 `SHA256SUMS.txt` 中对应文件的值比较；完全一致再安装。

## 2. 第一次使用

首次引导会询问以下内容：

1. **昵称与身份**：用于调整首页称呼和内容风格，仅保存在本机。
2. **陪伴语气**：可选择更温柔、幽默、直接或鼓励的表达。
3. **隐私权限**：决定是否记录应用名称、窗口标题和电脑空闲状态。
4. **首个目标**：填写今天最想推进的一件事，也可以稍后再添加。

建议保留“应用活动记录”和“空闲检测”，并让“窗口标题记录”保持关闭。这样既能得到时间回顾，也能减少敏感信息进入本地记录。

## 3. 认识主界面

![DayMate 今日主界面](docs/screenshots/daymate-dashboard-v0.2.0.png)

左侧导航用于切换主要功能：

- **今日**：问候、今日三件事、任务建议和精神补给。
- **时间回顾**：查看应用使用时长、键鼠活跃情况和时间线。
- **任务**：添加、完成、删除任务，并设置优先级、预计时间和截止日期。
- **每日内容**：单独切换好句、音乐、轻松内容和微挑战。
- **数据与隐私**：暂停记录、调整隐私开关或删除活动数据。
- **设置**：主题、浮动球、音乐偏好和 AI 服务配置。

每天的柔和背景会按日期自动更换，也可以在界面中手动换景。同一天自动选择的背景默认保持一致。

## 4. 今日任务与专注

### 添加任务

进入“今日”或“任务”，填写任务名称，并按需要设置：

- 优先级
- 预计完成时间
- 截止日期

首页最多突出显示三项重要任务。“帮我选一件先做”即使没有启用 AI 也能使用，它会优先考虑截止日期、优先级和可快速完成的任务。

### 开始专注

1. 在任务卡片上点击开始。
2. 选择专注时长。
3. 倒计时期间可以暂停、继续或提前完成。
4. 完成后返回首页查看任务状态和今日专注时间。

如果暂时没有状态，可以先用 5 分钟启动。DayMate 的目标是帮你更容易开始，不会锁定电脑或强制关闭其他软件。

## 5. 音乐推荐与播放

DayMate 会根据时间、未完成任务数量和已选择的音乐偏好推荐歌曲。音乐直接在应用内播放，切换 DayMate 页面或收起主窗口后不会停止。

### 搜索与分类

- 可选择智能、专注、国风民乐、古典、自然或电子等类别。
- 搜索框支持歌曲名、歌手名或组合搜索，例如“张杰 天下”。
- 在线搜索使用 Audius 开放音乐目录，能否找到某首商业歌曲取决于该曲库是否收录并允许播放。
- 搜不到时可换用歌手名、英文名或关键词，也可以导入自己电脑中的本地音频。
- 网络曲库不可用时，应用会回退到随安装包提供的 CC0 离线音乐，不影响任务等其他功能。

### 播放按钮

- **自动连播**：当前歌曲结束后继续播放下一首推荐。
- **随机推荐**：随机选择下一首歌曲。
- **顺序播放**：按照当前列表顺序播放。
- **单曲循环**：当前歌曲结束后重新播放同一首。
- **下一首**：立即切换歌曲。

音乐类别只影响音乐推荐，不会联动更换每日好句、轻松内容或微挑战。

## 6. 浮动球与后台运行

![DayMate 桌面浮动球](docs/screenshots/daymate-floating-ball.png)

- 点击侧边栏的“收起为浮球”，主窗口会隐藏，桌面上保留浮动球。
- 按住浮动球拖动，可把它放到顺手的位置。
- 左键单击浮动球，恢复主窗口。
- 音乐播放时，浮动球显示律动柱。
- 右键单击浮动球，可显示主窗口、隐藏浮球或完全退出 DayMate。

关闭主窗口后，DayMate 可能仍在系统托盘中运行，以继续记录时间和播放音乐。这是正常行为。

## 7. 系统托盘与退出软件

系统托盘位于 Windows 任务栏右下角。DayMate 图标可能收在“显示隐藏的图标”箭头中。

- **左键单击托盘图标**：打开 DayMate 主窗口。
- **右键单击托盘图标**：打开原生菜单。
- 在右键菜单中选择“退出 DayMate”：完全结束主窗口、浮动球、音乐和后台记录。

鼠标只是移到托盘图标上不会弹出主窗口。若要完全退出，请使用浮动球或托盘的右键菜单，不需要任务管理器。

## 8. 时间回顾与键鼠活跃判断

![DayMate 时间回顾](docs/screenshots/daymate-review-v0.3.0.png)

时间回顾会显示：

- 电脑活跃时长
- 应用使用时长和时间线
- 鼠标点击次数
- 键盘按键次数
- 最近一次键鼠输入时间
- 空闲时间

DayMate 使用 Windows 的“最近一次键鼠输入时间”判断电脑是否真正处于使用状态。连续 5 分钟没有输入后，亮屏时间不会继续计入活跃时长，从而减少电脑长期开着但人已离开的误判。

为保护隐私，键鼠统计只保存次数，不保存：

- 具体按键内容
- 密码或聊天内容
- 鼠标坐标
- 点击目标
- 屏幕截图

## 9. AI 服务配置（可选）

没有 AI Key 时，任务、统计、每日内容、音乐搜索、背景和本地规则推荐仍可正常使用。

如需 AI 增强：

1. 打开“设置”中的“AI 服务”。
2. 开启“AI 增强”。
3. 选择服务商，例如商汤日日新、OpenAI、DeepSeek、通义千问、智谱、Moonshot、OpenRouter、Ollama 或自定义 OpenAI 兼容接口。
4. 填写服务商提供的 Base URL、模型名称和 API Key。
5. 点击“测试连接”，成功后保存。

API Key 保存在 Windows 凭据管理器，不会写入 Git 仓库、普通配置、SQLite 活动数据库或日志。不要把自己的 Key 发到 GitHub Issue、截图或公开聊天中。

AI 音乐推荐只发送必要的汇总信息，例如活跃分钟、未完成任务数量和音乐偏好；不会默认发送窗口标题、任务正文或聊天内容。

## 10. 数据与隐私

DayMate 默认采用本地优先设计：

- 活动记录保存在本机 SQLite 数据库。
- 任务和偏好保存在本机 WebView 数据中。
- 不要求账号，不提供活动数据云同步。
- 窗口标题记录默认关闭。
- 不记录键盘内容、不截屏、不读取聊天或文档正文。

正式安装版数据库通常位于当前 Windows 用户的应用数据目录：

```text
%APPDATA%\com.daymate.desktop\daymate.sqlite3
```

实际路径可能随 Tauri 或 Windows 环境变化。删除数据前建议先完全退出 DayMate，并在“数据与隐私”页面优先使用应用内提供的删除操作。

开发仓库自带的 D 盘启动方式会把开发数据库保存到：

```text
D:\DayMate\data
```

完整隐私说明请阅读 [PRIVACY.md](PRIVACY.md)。

## 11. 更新软件

1. 打开 [GitHub 最新版本页面](https://github.com/zhangweiguo9719-web/daymate-desktop/releases/latest)。
2. 阅读版本更新说明。
3. 下载最新的 `DayMate_x.y.z_x64-setup.exe`。
4. 完全退出正在运行的 DayMate。
5. 运行新安装包完成覆盖安装。

更新前通常不需要卸载旧版本。涉及数据迁移时，请以对应 Release 的说明为准。所有正式更新纪要也可在 [CHANGELOG.md](CHANGELOG.md) 查看。

## 12. 卸载软件

1. 先通过托盘或浮动球右键菜单完全退出 DayMate。
2. 打开 Windows“设置 → 应用 → 已安装的应用”。
3. 找到 DayMate，选择“卸载”。

如果希望连同本地活动历史一起删除，请先在 DayMate 的“数据与隐私”页面清空数据。卸载前也可以自行备份数据库。

## 13. 常见问题

### 安装后双击没有反应

先检查系统托盘，应用可能已经在后台运行。仍无法打开时，重启 Windows 后再试，并在 GitHub Issue 中提供 Windows 版本、DayMate 版本和可复现步骤。

### 音乐提示“暂时走丢了”

这通常表示在线曲库临时不可用或当前搜索无结果。可以稍后重试、换关键词、切换类别或导入本地歌曲；任务和时间记录不会受影响。

### 搜不到想听的商业歌曲

DayMate 当前使用开放音乐目录，不绕过版权、会员或平台登录限制，因此无法保证收录所有商业歌曲。可尝试歌手名加歌曲名，或使用自己合法持有的本地音频。

### 切换窗口后音乐停止

0.3.0 起播放器会常驻应用生命周期。请确认没有在托盘菜单中完全退出 DayMate，并升级到最新版本。如果仍能复现，请提交 Issue。

### 为什么关闭窗口后进程还在

DayMate 会最小化到托盘，以继续活动统计、音乐和浮动球。需要结束进程时，右键托盘或浮动球并选择“退出 DayMate”。

### 时间统计为什么比亮屏时间短

连续 5 分钟没有键鼠输入后会被判定为空闲，避免把人已离开但屏幕未熄灭的时间算作活跃使用。

### 如何报告问题

前往 [GitHub Issues](https://github.com/zhangweiguo9719-web/daymate-desktop/issues) 提交问题，并附上：

- Windows 版本
- DayMate 版本
- 问题现象与复现步骤
- 预期结果和实际结果
- 必要的界面截图

提交前请遮挡 API Key、窗口标题、任务内容、用户名和个人路径等隐私信息。
